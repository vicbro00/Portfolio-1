## Getting Started

### Installing

Clone the repo

1. Clone the repo:

```bash
git clone git@https://github.com/vicbro00/Semester-Project.git
```

2. Install the dependencies:

```
npm install
```

### Running

Nothing special needed to run as it's only HTML and CSS.

To run the app, run the following commands:

```bash
npm run start
```
